# VoiceLocal 🎙️

Push-to-talk голосовой ввод текста на локальном Whisper.
Работает **полностью офлайн**, без облаков. Windows 10/11.

Проект вырос из двух независимых разработок:
- **[Maxim Gurskii](https://github.com/mmxm162)** (https://github.com/mmxm162) написал PyQt6 интерфейс с тёмной темой, реализовал Toggle и Hold режимы PTT, добавил голосовые команды ("точка", "запятая", "новая строка" и др.) и использовал sounddevice для захвата аудио.
- **[Рамиль Мохнатти](https://github.com/Mokhnatti)** добавил ring buffer (захват 1.5 сек до нажатия), портативный Python, CUDA поддержку, мут других микрофонов во время записи, кнопку копирования результата и кнопку перезапуска.

## Установка

### 1. Установить Python 3.11+

Скачать с [python.org](https://python.org). При установке отметить **"Add Python to PATH"**.

### 2. Установить зависимости

```
pip install -r requirements.txt
```

### 3. Запустить

```
python main.py
```

Или двойным кликом на `VoiceLocal.bat`.

При первом запуске Whisper автоматически скачает модель (~1.5 ГБ). Это происходит один раз.

## Возможности

- **Toggle** или **Hold** режим PTT — на выбор в настройках
- **Ring buffer** — захватывает 1.5 сек до нажатия кнопки
- **CUDA** (NVIDIA) — автоматически, fallback на CPU
- **Голосовые команды** — "точка", "запятая", "новая строка" и др.
- **Мут других микрофонов** — во время записи все лишние микрофоны замолкают
- **Фильтр галлюцинаций** Whisper
- Красивый тёмный UI с иконкой в трее

---

## Использование

1. Программа запускается в **системном трее** (правый нижний угол)
2. Поставь курсор в любое поле ввода
3. **Toggle (по умолчанию):** нажми `Scroll Lock` → говори → нажми ещё раз
4. **Hold:** зажми `Scroll Lock` → говори → отпусти
5. Текст вставится автоматически

## Настройки

Двойной клик по иконке трея → ⚙ кнопка.

| Параметр | Описание |
|----------|----------|
| Режим PTT | Toggle или Hold |
| Клавиша | Scroll Lock, Home, Pause, F13... |
| Модель | tiny / base / small / medium / large-v3-turbo |
| CUDA | GPU ускорение (только NVIDIA) |
| Мут других микрофонов | Мутит всё кроме выбранного во время записи |
| Голосовые команды | "точка", "запятая", "новая строка"... |
| Auto Enter | Enter после вставки |

## Модели Whisper

| Модель | Размер | Скорость | Качество |
|--------|--------|----------|----------|
| tiny | 75 МБ | очень быстро | базовое |
| base | 145 МБ | быстро | среднее |
| small | 465 МБ | средне | хорошее |
| medium | 1.5 ГБ | умеренно | отличное |
| large-v3-turbo | 1.5 ГБ | быстро | **лучшее** ★ |
| large-v3 | 3 ГБ | медленно | максимум |

## Структура проекта

```
VoiceLocal/
├── main.py
├── VoiceLocal.bat
├── requirements.txt
├── config/
│   └── settings.json       # Настройки (создаётся автоматически)
└── app/
    ├── config.py            # Настройки + константы
    ├── controller.py        # PTT логика (toggle/hold + ring buffer)
    ├── hotkey.py            # Глобальный перехват клавиш
    ├── recorder.py          # Запись аудио + ring buffer
    ├── transcriber.py       # Whisper (CUDA/CPU + постобработка)
    ├── inserter.py          # Вставка текста
    ├── audio_mute.py        # Мут микрофонов (Windows API)
    ├── main_window.py       # Главное окно (PyQt6)
    ├── settings_dialog.py   # Диалог настроек
    └── tray.py              # Иконка в трее
```
