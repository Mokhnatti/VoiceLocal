# VoiceLocal 🎙️

Push-to-talk голосовой ввод текста на локальном Whisper.
Работает **полностью офлайн**, без облаков. Windows 10/11.

> Совместная разработка: Рамиль Мохнатти + Максим Гурский

---

## Возможности

- **Toggle** или **Hold** режим — на выбор в настройках
- **Ring buffer** — захватывает 1.5 сек до нажатия кнопки
- **CUDA** (NVIDIA) — автоматически, с fallback на CPU
- **Голосовые команды** — "точка", "запятая", "новая строка" и др.
- **Фильтр галлюцинаций** Whisper
- Портативный Python — папка `python/` (ничего не устанавливать)
- Красивый тёмный UI с иконкой в трее

---

## Установка

### Вариант 1 — Портативный (рекомендуется)

Скачай релиз с папкой `python/` — просто распакуй и запусти `VoiceLocal.bat`.
Python и все библиотеки уже внутри. Ничего не нужно устанавливать.

> Для AMD / без GPU: удали папку `cuda/` перед передачей другу.

### Вариант 2 — Системный Python

```bash
pip install -r requirements.txt
python main.py
```

Требуется Python 3.11+. Модель (~1.5 ГБ) скачается при первом запуске.

## Использование

1. Программа запускается в **системном трее** (правый нижний угол)
2. Поставь курсор в любое поле ввода
3. **Toggle (по умолчанию):** нажми `Scroll Lock` → говори → нажми ещё раз
4. **Hold:** зажми `Scroll Lock` → говори → отпусти
5. Текст вставится автоматически

## Настройки

Двойной клик по иконке трея → открыть окно → ⚙ кнопка.

| Параметр | Описание |
|----------|----------|
| Режим PTT | Toggle или Hold |
| Клавиша | Scroll Lock, Home, Pause, F13... |
| Модель | tiny / base / small / medium / large-v3-turbo |
| CUDA | GPU ускорение (только NVIDIA) |
| Голосовые команды | "точка", "запятая", "новая строка"... |
| Auto Enter | Enter после вставки |

## CUDA (GPU ускорение)

Папка `cuda/` **не включена** в репозиторий (слишком большая). Без неё работает на CPU.

Чтобы добавить CUDA:
1. Скачай [CUDA Toolkit 12.x](https://developer.nvidia.com/cuda-downloads) от NVIDIA
2. Или скопируй из существующей установки нужные DLL в папку `cuda/`:
   - `cublas64_12.dll`, `cublasLt64_12.dll`
   - `cudart64_12.dll`
   - `cudnn_ops_infer64_8.dll` и др.
3. VoiceLocal автоматически подхватит их при запуске

> Если NVIDIA карты нет — папка `cuda/` не нужна, программа сама переключится на CPU.

## Автозапуск

`Win + R` → `shell:startup` → создай ярлык на `VoiceLocal.bat`
(или используй `launch.vbs` для запуска без окна консоли)

## Структура проекта

```
VoiceLocal/
├── main.py                  # Точка входа
├── VoiceLocal.bat           # Запуск (portable или system python)
├── launch.vbs               # Тихий запуск без консоли
├── requirements.txt
├── python/                  # Портативный Python (опционально)
├── model/                   # Модель Whisper (опционально, иначе скачается)
├── cuda/                    # CUDA DLL для NVIDIA (опционально)
├── config/
│   └── settings.json        # Настройки (создаётся автоматически)
└── app/
    ├── config.py            # Настройки + константы
    ├── controller.py        # PTT логика (toggle/hold + ring buffer)
    ├── hotkey.py            # Глобальный перехват клавиш
    ├── recorder.py          # Запись аудио + ring buffer
    ├── transcriber.py       # Whisper (CUDA/CPU + постобработка)
    ├── inserter.py          # Вставка текста
    ├── main_window.py       # Главное окно (PyQt6)
    ├── settings_dialog.py   # Диалог настроек
    └── tray.py              # Иконка в трее
```
